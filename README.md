# ABOUT ME

- 💻 **Chief Technology Officer** @ [Vaga Beta](https://www.vagabeta.rs/)

## 🚀 What I'm Working On

- **eVaga MKO Firmware**: Bare-metal BLE firmware for a Nordic nRF52810 (Cortex-M4) smart-weighing verification device: nRF5 SDK 17.1, SoftDevice S112, custom GATT services, Ynvisible e-ink display, PCF8563 RTC, FDS flash storage, and ultra-low-power design
- **eVaga MKO Operator App**: Cross-platform Flutter (Android + iOS) companion for the metrology module, a BLE central that sets verification deadlines, syncs the on-device RTC, drives the power-cut relay, and monitors live status; Riverpod + clean architecture, PIN/biometric app-lock, and a Cloudflare-backed scale registry
- **eVaga Desktop**: Windows WinForms suite (.NET 8 / C# 12) for industrial truck-scale weighing. A server app bound to the scale indicator over serial (RS232) backed by SQL Server and a Fleck WebSocket server, plus a thin client terminal that pulls everything over WebSocket for display and weighing-slip printing; layered architecture (Repository + Unit of Work + Service + manual DI), AES-encrypted sessions, BCrypt auth, and cloud licensing + auto-updates over HTTPS to Firebase Cloud Functions
- **Vaga Beta Platform**: Advanced React e-commerce application with Firebase backend [vagabeta.rs](https://vagabeta.rs)
- **Radar Puls**: Community Traffic Reporting Platform [radarpuls.com](https://radarpuls.com)
- **Electronics Projects**: PCB design using Altium Designer Pro
- **Smart Weighing Systems**: Integrating hardware with software solutions

## 🎯 Specializations

- **Embedded Firmware**: Bare-metal C on ARM Cortex-M (Nordic nRF52), Bluetooth LE GATT services, low-power design, and on-device drivers (I²C/TWI, RTC, e-ink, PWM)
- **Cross-Platform Mobile**: Flutter/Dart apps with Riverpod state management, feature-first clean architecture, go_router navigation, and a custom glassmorphism design system
- **BLE Central / Companion Apps**: flutter_blue_plus GATT clients: service discovery, BCD time sync, notification streams, and bonded pairing for embedded devices
- **E-commerce Development**: Full-stack solutions with advanced cart systems and user authentication
- **Firebase Integration**: Authentication, Firestore, Storage, and App Check implementation
- **PCB Design**: Professional PCB development using Altium Designer
- **Responsive Design**: Mobile-first approach with smooth animations and modern UI/UX
- **Hardware-Software Integration**: Connecting weighing systems with web applications

### 🏆 Featured Projects

## 🛒 Vaga Beta E-commerce Platform

- **Tech Stack**: React, Firebase, Tailwind CSS, Vite
- **Features**: Advanced cart system, user authentication, product management, responsive design
- **Highlights**: Smooth animations, progressive image loading, admin panel

### 📡 eVaga MKO BLE Firmware

- **Tech Stack**: C, nRF5 SDK 17.1, SoftDevice S112, ARM Cortex-M4 (nRF52810), J-Link/RTT
- **Features**: Custom BLE GATT services, MITM-bonded pairing, Ynvisible e-ink display, PCF8563 RTC time sync, FDS flash persistence, watchdog + brown-out mitigation
- **Highlights**: Ultra-low-power (DCDC + parked GPIO), bus-recovery I²C, layered driver architecture, Unity-tested calendar logic

### 📱 eVaga MKO Operator App

- **Tech Stack**: Flutter, Dart 3, Riverpod, go_router, Freezed, flutter_blue_plus, Cloudflare (Workers + D1 + R2)
- **Features**: BLE central for the metrology module: RTC time sync, verification-deadline & period scheduling, power-cut relay control, and a live status countdown over GATT notifications
- **Highlights**: Feature-first clean architecture, dark "luxury" glassmorphism UI (Royal Blue + champagne gold), PIN + biometric app-lock, offline cache, cloud scale registry, golden + Patrol tests, and a mock-BLE demo mode

### 🖥️ eVaga Desktop

- **Tech Stack**: C# 12, .NET 8, WinForms, SQL Server, Fleck WebSocket, ADO.NET, BCrypt, ClosedXML, AES, Firebase Cloud Functions
- **Features**: Two-app server/client architecture over WebSocket (port 5000), serial scale-indicator drivers (DiniArgeo, Birotehna, A12), measurement evidence, returnable-packaging rental tracking, weighing-slip printing, and cloud-backed license activation/verification + auto-updates
- **Highlights**: Clean layered architecture (Repository + Unit of Work + Service + Factory), thread-safe WinForms UI, fully parametrized SQL, AES-encrypted WebSocket sessions, role-based access (BCrypt), hardware-ID licensing, and a Firebase Functions backend for licensing and update feeds

### ⚖️ Smart Weighing Systems

- **Integration**: Hardware weighing scales with software solutions
- **Applications**: Industrial and educational measurement systems
